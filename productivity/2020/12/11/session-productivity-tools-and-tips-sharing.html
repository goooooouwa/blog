入门级效率工具和快捷键分享

Dec 11, 2020

时间管理

  1. Tip: setup regular activities
  2. Tip: block focus time
  3. Menu bar calendar: itsycal
  4. On screen timer: timer app
  5. World timezone: Menu World Time

任务管理

  1. Global shortcut to quickly add a todo item (e.g. ⌃+⇧+N for 滴答清单)
  2. Voice command to create a timely reminder (Say: “Hey Siri, remind me to buy milk at 4pm”)

笔记管理

  1. Global shortcut to quickly add a todo item (e.g. ⌘+⌃+N for 印象笔记)
  2. 定期review inbox （印象笔记App整理助手）
  3. Keyboard shortcuts to organize notes（⌘+J跳转到folder，⌘+⌃+M 移动到folder）

文件管理

  1. Finder导航：进入folder，打开文件，切换视图（⌘+1/2/3/4）
  2. 截图快捷键：screenshot & paste (⌘+⇧+4/5, ⌘+⌃+⇧+4/5)
  3. 标记图片：Preview
  4. 创建图示：Preview / Keynote / draw.io
  5. 录屏生成Gif动图：RecordIt

窗口管理

  1. 打开Dock App快捷键：Snap
  2. 打开目录快捷键：Desktop folder (⌘+⇧+D), Downloads folder (⌘+⌥+L)
  3. Split screen: ⌘+⌥+⌃+←/→ with Shiftit
  4. 显示器间移动窗口：⌘+⌥+⌃+N/P with Shiftit
  5. 全屏切换：⌃+←/→
  6. Reveal Desktop（显示桌面）: four fingers spread
  7. App Expose（显示当前应用所有窗口）: three fingers down
  8. 锁屏：⌃+⌘+Q

密码管理

  1. 1Password 打开mini窗口快捷键：⌘+⌥+\
  2. 快速copy username & password：←/→/↑/↓+⏎

文本编辑

  1. Basic: 输入法文本替换（System Preferences -> Keyboard -> Text）
  2. Advanced: dynamic snippets with Alfred
  3. Clipboard history with CopyClip
  4. Markdown editor Typora
    1. markdown slides with Marp
  5. 永不中断的语音输入 with Drafts

快捷操作

  1. Basic: Spotlight
  2. Advanced: Alfred
    1. 找app：直接输入App名字
    2. 找文件（including Google Drive）：输入空格然后输入文件名字
    3. 打开浏览器书签

Mac Keyboard Symbols explained

  • ⌘ means Command
  • ⌥ means Option (also called “Alt”)
  • ⌃ means Control
  • ⇧ means Shift
  • ⌫ means Delete (called Backspace on Windows keyboards)
  • ⌦ means Forward Delete (called Delete on Windows keyboards)
  • ⏎ means Return (also called “Enter”)
  • ⎋ means Escape
  • ⇥ means Tab right
  • ⇤ means Tab left
  • ⇪ means Caps lock
  • ⏏ means Eject
  • ↑ means Up
  • ↓ means Down
  • ← means Left
  • → means Right
  • ⇞ means Page Up
  • ⇟ means Page Down
  • ↖︎ means Top (called Home on Windows computers)
  • ↘︎ means End
