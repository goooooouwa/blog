--- layout: post title: 'PhoneGap project directory analysis' categories: phonegap --- PhoneGap project directory analysis
There’re four folders under a newly created PhoneGap project:
merges    platforms plugins   www 

platforms is where the platform code is, in which the www folder is a mirror of master code under www.
www is where the master code is, which will override code under platforms/[platform name]/www, when compiling.
