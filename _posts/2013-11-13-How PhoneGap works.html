--- layout: post title: 'How PhoneGap works' categories: phonegap --- How PhoneGap works
PhoneGap essentially uses url as the common language between Native code and Javascript code. Because both Native code and Javascript code can access the url value.

PhoneGap uses XHR and iframe( fallback option) to implement this solution. See https://github.com/phonegap/phonegap/blob/master/lib/ios/CordovaLib/cordova.js#L983

Beside this, PhoneGap also needs to convert API response into json format.
