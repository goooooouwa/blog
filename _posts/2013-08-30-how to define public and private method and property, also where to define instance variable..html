--- layout: post title: 'how to define public and private method and property, also where to define instance variable.' categories: objective-c --- how to define public and private method and property, also where to define instance variable.
Note: There're three things in an object: property, method and instance variable. The difference between property and instance variable is instance variable has no getter and setter so it can not be accessed from outside , and the ARC thing. You should always access the instance variables directly from within an initialization method because at the time a property is set, the rest of the object may not yet be completely initialized.


// in .h file

@interface className : parentClassName {

//instance variables declaration can go here
type *instanceVariable;


}

//public properties and methods declaration goes here
@property type *publicProperty;
-(type)publicMethod;

@end




// in .m file

@interface className () {

//instance variables declaration can also go here
type *AnotherinstanceVariable;

}

//private properties and methods declaration goes here
@property type *privateProperty;
-(type)privateMethod;

@end

@implementation

//implementation for public and private methods go here

@end
