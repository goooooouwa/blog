--- layout: post title: 'Asynchronous XMLHttpRequest implementation' categories: javascript programming paradigm --- Asynchronous XMLHttpRequest implementation
Quote from Specification of the XMLHttpRequest object for Mozilla developers:
async
If true, notification of a completed transaction is provided using event listeners. This must be true if the multipart attribute is true, or an exception will be thrown.

Asynchronous XMLHttpRequest’s implementation is based on events. So what's behind the asynchronous process is event-driven programming.
