# Jump to closing bracket in Visual Studio Code

VS code 收起花括号的键盘快捷键

**Mac** `Cmd+Shift+\`

**Windows** `Ctrl+Shift+\`

As well you can do:

`Ctrl+Shift+p`

And select

`Preferences: Open Keyboard Shortcuts`

There you will be able to see all the shortcuts, and create your own.
