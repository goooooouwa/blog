--- layout: post title: 'Web Service API Authentication' categories: Authentication HTTPS web service --- Web Service API Authentication
1. client let user provide username and password via HTTP basic authentication over HTTPS
2. if correctly matched, server responds with a temporary token
3. from now on, client can use the token with every HTTP request to communicate with the server


Authentication methods Rails support:
Basic, Digest and Token Authentication: Rails comes with out-of-the-box support for three kinds of HTTP authentication.
