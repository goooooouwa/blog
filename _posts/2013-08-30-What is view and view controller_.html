--- layout: post title: 'What is view and view controller?' --- What is view and view controller?
A view object defines a rectangular region on the screen and handles the drawing and touch events in that region. it cares only about things related to the representation of the visual area. E.g. a textView has a text property which stores the text to show in it's visual area.

A content view controller presents content on the screen using a view or a group of views organized into a view hierarchy. A container view controller contains content owned by other view controllers. E.g. a tab bar view controller manages a set of view controllers designated to the tabs.

A segue represents a visual transition from one scene to another.
