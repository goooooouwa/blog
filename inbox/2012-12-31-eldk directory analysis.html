sysroot/
<host>/: everything needed to build and run qt programs on host, like qmake, uic and moc for host.
usr/bin/<target>/: cross tool-chain for host to target.
<target>/: everything needed to build and run qt programs on target, like qmake, uic and moc for target.
environment-setup-<target>: environment variables for target.
site-config-<target>: site configuration for target.
version-<target>: version of eldk for target.
rootfs-<type>/: root filesystem to install on target device, for purpose on <type>.

p.s. <host> example: i686-eldk-linux, <target> example: arm-linux-gnueabi, <type> example: lsb, see Supported Target Configurations. 
The Embedded Linux Development Kit (ELDK) includes the GNU cross development tools, such as the compilers, binutils, gdb, etc., and a number of pre-built target tools and libraries necessary to provide some functionality on the target system.
