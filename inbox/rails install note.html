In short: all you need is RVM.
dependencies:
RVM <= Ruby, RubyGems 
Ruby <= RubyGems
RubyGems <= Rails
RubyGems is now part of the standard library from Ruby version 1.9.

In fact, if you have installed RVM, you already have RubyGems, since RVM includes it automatically:

$ which gem
/Users/mhartl/.rvm/rubies/ruby-1.9.3-p0/bin/gem

Note command 'gem' is actually RubyGems.

Metaphors: Bundler is like APT( dependencies manager. Gemfile is the dependency file), RubyGems is like dpkg( package manager. gem ~> .deb package), and Rake is like "make"( build tool. rakefile ~> makefile).
