This method combines multiple promises into a single promise. If all of the input promises are resolved successfully, this resulting promise will be resolved with an array of promises. If any of the promises is rejected, the resulting promise will be rejected with the same rejection value the rejected promise returns.

  • all(promises);

    Combines multiple promises into a single promise that is resolved when all of the input promises are resolved.

    Parameters

    Param Type Details
    promises Array.<Promise>Object.<Promise>

    An array or hash of promises.

    Returns

    Promise

    Returns a single promise that will be resolved with an array/hash of values, each value corresponding to the promise at the same index/key in the promises array/hash. If any of the promises is resolved with a rejection, this resulting promise will be rejected with the same rejection value.
