Step 1: Obtaining a request token
a. configure a callback url in your app
b. use Twitter’s OAuth tool to auto-generate cURL command( replace the “Request URI” entirely)
c. run the command in Terminal
Step 2: Redirecting the user
a. copy&paste the oauth_token from command output into the redirecting URL
b. user sign in and authorize the app
Step 3: Converting the request token to an access token
a. copy&paste the oauth_token & oauth_verifier found in redirected URL into Twitter OAuth tool settings( oauth_token in "Access token" and oauth_verifier in "Request query”)
b. auto-generate the cURL command and run it in Terminal
