There are two formats for .ruby-version file:
  1. version number with "ruby-“ prefix, like “ruby-2.1.2"
  2. version number without “ruby-“ prefix, like “2.1.2"

The support of these two formats of the major ruby version managers are as following:
rbenv only supports “2.1.2” supports both “2.1.2” and "ruby-2.1.2"
rvm supports both “2.1.2” and "ruby-2.1.2"
RubyMine only supports "ruby-2.1.2” for now

UPDATE:
rbenv now supports "ruby-2.1.2" as well. https://github.com/sstephenson/rbenv/commit/6820704c91afdd875e2a11bc647659c4d557aa5d
RubyMine will support in version 7.1. https://youtrack.jetbrains.com/issue/RUBY-15058

So I recommend we use “ruby-2.1.2” format in our .ruby-version file.
